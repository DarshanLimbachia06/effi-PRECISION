# effiPrecision: High-Fidelity Performance Metrology Dashboard

**effiPrecision** is a premium, professional-grade technical inspection instrument designed for modern HR and performance management. It moves beyond traditional "soft" metrics into the realm of **behavioral metrology**, applying engineering principles such as GD&T (Geometric Dimensioning and Tolerancing) and SPC (Statistical Process Control) to human performance data.

## 🚀 Key Features

- **Sensor Matrix**: A high-density data visualization grid performing multi-sensor fusion across Jira, GitHub, and Slack telemetry.
- **SPC Orchestrator**: Real-time Statistical Process Control charts for detecting behavioral drift and calibrating manager rating variance.
- **GD&T Tolerancing Engine**: Behavioral "tolerancing" with LMC (Least Material Condition) and MMC (Maximum Material Condition) adjustments for performance flexibility.
- **Cross-Sensor Duel**: Comparative analytics between different engineering sensors to identify alignment or dissonance in team workflows.
- **AI-Powered Precision Assistant**: A data-aware agentic interface for querying deep-level performance insights using natural language.

## 🛠️ Tech Stack

- **Frontend**: React.js, Vite, TailwindCSS (Onyx & Electric Violet Premium Aesthetic).
- **Backend**: Python (FastAPI/Flask), Agentic Logic.
- **Data Engine**: Multi-source telemetry fusion.
- **Design Concepts**: Glassmorphism, Metrology-inspired UI, High-contrast visualization.

## 📦 Installation & Setup

### 1. Environment Configuration
Ensure you have a `.env` file in the root with the necessary API keys (OpenAI/Gemini, etc.).
```env
OPENAI_API_KEY=your_key_here
```

### 2. Backend Setup
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python api.py
```

### 3. Frontend Setup
```bash
cd web-ui
npm install
npm run dev
```
